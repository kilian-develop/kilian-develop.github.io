## 🛠️ 기술 스택

### 주력 기술
- **Java** - 메인 개발 언어
- **Spring Boot / Spring Framework** - 엔터프라이즈 애플리케이션 개발
- **JPA / Hibernate** - ORM 기술

### 관심 분야
- **도메인 주도 설계 (DDD)** - 비즈니스 로직 중심 설계
- **클린 아키텍처** - 계층 분리와 의존성 관리
- **마이크로서비스** - 분산 시스템 아키텍처
- **데이터 처리** - Kafka, Spring Batch, MariaDB, MongoDB
- **대규모 데이터 처리** - 성능 최적화와 확장성

### 기타 기술
- Kafka - 메시지 스트리밍 및 이벤트 처리
- Spring Batch - 대용량 데이터 처리
- MariaDB - 관계형 데이터베이스
- MongoDB - NoSQL 데이터베이스
- Docker - 컨테이너 기술
- Git - 버전 관리
